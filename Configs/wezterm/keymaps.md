# WezTerm Keymaps Cheat Sheet

## Leader
CTRL + a

---

## Utility
Leader+[        → copy mode  
Leader+:        → command palette  
Leader+t        → change color scheme  
Leader+Ctrl+a   → send Ctrl+a  

---

## Tabs
Leader+c        → new tab  
Leader+n        → next tab  
Leader+p        → prev tab  
Leader+w        → tab navigator  
Leader+,        → rename tab  
Leader+.        → move tab mode  
Leader+1-9      → go to tab  

### Move Tab Mode (Leader+.)
h/j             → move left  
k/l             → move right  
Enter/Esc       → exit  

---

## Panes
Leader+-        → split vertical  
Leader+|        → split horizontal  
Leader+x        → close pane  
Leader+Space    → rotate panes  
Leader+!        → pane → new tab  
Leader+m        → zoom  

### Resize (fast)
Leader+h/j/k/l  → resize (big)  

### Resize Mode (Leader+r)
h/j/k/l         → resize (small)  
Enter/Esc       → exit  

---

## Move Panes (Leader+Ctrl+m)
h               → rotate CCW  
l               → rotate CW  
s               → swap (select)  
Enter/Esc       → exit  

---

## Navigation (smart)
Ctrl+h/j/k/l    → move between panes  

---

## Sessions / Workspaces (Leader+s)
w               → switch workspace (fuzzy)
s               → new workspace (prompt name)
r               → rename current workspace
[               → next workspace
]               → prev workspace
p               → workspace "main"
Enter/Esc       → exit
